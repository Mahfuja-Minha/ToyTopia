# ToyTopia

## Project Purpose
ToyTopia is a toy marketplace web application where users can explore, search, filter, and sort toys by category and price. The project is built to practice modern React development with routing, state management, reusable components, and responsive UI design.

---

## Live Website
Live URL: https://toytopia9.netlify.app/


---

## Key Features
- Browse all toys with a clean and responsive UI
- Category-wise toy filtering using dynamic routes
- Search toys by name
- Sort toys by price (Low → High, High → Low)
- Reusable toy card components
- Sticky category sidebar for better UX
- Loading spinner while data is fetching
- Fully responsive for mobile, tablet, and desktop
